# 📖 Project Overview

The Social Media Analytics Dashboard is an interactive Business Intelligence solution developed using Microsoft Power BI. The project transforms raw social media data collected from multiple platforms into meaningful insights that support better business decisions.
The dashboard enables users to monitor campaign performance, audience engagement, platform comparisons, and business growth through interactive visualizations and Key Performance Indicators (KPIs).

---

# 🎯 Project Objectives

* Analyze social media performance across multiple platforms.
* Monitor engagement, likes, comments, and audience growth.
* Compare marketing campaign performance.
* Identify top-performing content and trends.
* Support data-driven decision making.
* Provide interactive dashboards for business users.

# ⚠️ Challenges

During the project, we faced several challenges:

* Collecting data from multiple social media platforms.
* Cleaning inconsistent and duplicate records.
* Handling missing values.
* Building an efficient data model.
* Designing an easy-to-use interactive dashboard.

---

# ✅ Solutions

To overcome these challenges, we:

* Cleaned and transformed the datasets using Power Query.
* Built relationships using Data Modeling.
* Created dynamic KPIs with DAX.
* Designed user-friendly dashboards with interactive visualizations.
